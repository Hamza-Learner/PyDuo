# 🚀 Getting Started

## Prerequisites

Install:

* Node.js 18+

---

## Clone the Repository

```bash
git clone https://github.com/Hamza-Learner/PyDuo.git
cd PyDuo
```

---

## Install Dependencies

```bash
npm install
```

---

## Configure Environment

Create a `.env` file in the project root.

```env
GEMINI_API_KEY=your_gemini_api_key_here
```

---

## Start Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

---

# 📦 Production

Build the project

```bash
npm run build
```

Start the production server

```bash
npm run start
```

---

# 📱 Android APK

Download the latest Android release from the project's GitHub Releases page.

> Signed APK ready to install.

Enable **Install Unknown Apps** on your Android device if prompted.

---

# ⚙️ Build Android APK (Capacitor)

## Install Capacitor

```bash
npm install @capacitor/core @capacitor/cli
```

## Initialize

```bash
npx cap init
```

Example

**App Name**

```text
PyDuo
```

**App ID**

```text
com.learning.pyduo
```

Set

```text
webDir = dist
```

---

## Add Android Platform

```bash
npm install @capacitor/android
npx cap add android
```

---

## Build & Sync

```bash
npm run build
npx cap sync
```

---

## Open Android Studio

```bash
npx cap open android
```

Generate a signed Release APK from:

**Build → Generate Signed Bundle / APK → APK**

---

# 🤖 GitHub Actions

Every version tag (`v*.*.*`) automatically builds and publishes a signed APK to GitHub Releases.
